# ProjectTag 组件说明

## 1. 组件定位
`ProjectTag` 是基于 `Badge` 组件封装的业务组件，用于在项目卡片或详情页中标识项目的垂类领域（如仿真、配管）。

## 2. 搭配方式与应用场景
- **项目卡片**：在 `ProjectCard` 内展示项目类型。
- **列表筛选**：在分类导航面板中作为视觉辅助。

## 3. 可修改的范围参数 (Props)
- `type` ('simulation' | 'piping' | string): 预设的业务类型或自定义字符串。
- `labelOverride` (string, 可选): 强制覆盖自动国际化的标签文本。
- `className` (string): 布局调整类名。

## 4. 设计规范禁忌（不建议修改）
- **样式变体**：强制使用 `Badge` 的 `pill` 变体，以保持白底阴影的“悬浮”视觉感。
- **色彩策略**：使用去饱和度的颜色（如 `#475569`, `#64748b`），避免喧宾夺主。
