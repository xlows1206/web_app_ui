# ProcAgent 项目推进清单

## 项目目标陈述
修复 Live View 与智能助手面板之间的拖拽交互问题，确保仅改变宽度而不产生缩放或位移。

## 关键模块描述
- **Workspace Layout**: 包含智能助手面板、Live View 面板及两者之间的可拖拽分隔条。
- **Resizer Logic**: 处理鼠标拖拽事件，计算并更新面板宽度的逻辑。

## 处理流程描述
1. 用户按下分隔条开始拖拽。
2. 计算鼠标位移并实时更新两侧面板的宽度。
3. 停止拖拽并保持最终宽度。
